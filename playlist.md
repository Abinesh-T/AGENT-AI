# AI Systems & Agent Engineering Video Curriculum

*A Structured YouTube Learning Path (Beginner → Advanced)*

Purpose:
This document complements the **AI Agent Systems course** you are learning through chat.
Use it during passive time (travel, breaks, evenings) to **reinforce concepts visually**.

Structure:

```
Layer 0 → Mental Models
Layer 1 → Programming Foundations
Layer 2 → Backend & System Design
Layer 3 → Distributed Systems
Layer 4 → AI / ML Foundations
Layer 5 → LLM & Transformer Models
Layer 6 → AI Engineering (RAG, Agents)
Layer 7 → AI Safety & Security
Layer 8 → Production AI Systems
Layer 9 → Finance Systems (ERP Thinking)
```

# How To Study With This Curriculum

Daily routine

```
30–60 minutes watching
10 minutes notes
5 minutes reflection
```

Ask yourself

```
What problem is this system solving?
What failure mode does it prevent?
How does it apply to AI systems?
```
